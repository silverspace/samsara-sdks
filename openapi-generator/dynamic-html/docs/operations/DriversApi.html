This is the API

createDriver

Create a new driver.

URL

https://api.samsara.com/v1/fleet/drivers/create

HTTP Method

POST

Response Type

Driver

Parameters

  • Query: access_token String

    Samsara API access token.

  • Body: createDriverParam DriverForCreate(DriverForCreate)

    Driver creation body

deactivateDriver

Deactivate a driver with the given id.

URL

https://api.samsara.com/v1/fleet/drivers/{driver_id_or_external_id}

HTTP Method

DELETE

Response Type

Parameters

  • Query: access_token String

    Samsara API access token.

  • Path: driver_id_or_external_id String

    ID of the driver. This must be either the numeric ID generated by Samsara or the external ID of the driver. External IDs are customer specified key-value pairs.

getAllDeactivatedDrivers

Fetch all deactivated drivers for the group.

URL

https://api.samsara.com/v1/fleet/drivers/inactive

HTTP Method

GET

Response Type

List

Parameters

  • Query: access_token String

    Samsara API access token.

  • Query: group_id Long

    Optional group ID if the organization has multiple groups (uncommon).
