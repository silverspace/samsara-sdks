This is the API

getDriverSafetyScore

Fetch the safety score for the driver.

URL

https://api.samsara.com/v1/fleet/drivers/{driverId}/safety/score

HTTP Method

GET

Response Type

DriverSafetyScoreResponse

Parameters

  • Path: driverId Long

    ID of the driver

  • Query: access_token String

    Samsara API access token.

  • Query: startMs Long

    Timestamp in milliseconds representing the start of the period to fetch, inclusive. Used in combination with endMs.

  • Query: endMs Long

    Timestamp in milliseconds representing the end of the period to fetch, inclusive. Used in combination with startMs.

getVehicleHarshEvent

Fetch harsh event details for a vehicle.

URL

https://api.samsara.com/v1/fleet/vehicles/{vehicleId}/safety/harsh_event

HTTP Method

GET

Response Type

VehicleHarshEventResponse

Parameters

  • Path: vehicleId Long

    ID of the vehicle

  • Query: access_token String

    Samsara API access token.

  • Query: timestamp Long

    Timestamp in milliseconds representing the timestamp of a harsh event.
