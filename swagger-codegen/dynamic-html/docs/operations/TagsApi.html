This is the API

createTag

Create a new tag for the group.

URL

https://api.samsara.com/v1/tags

HTTP Method

POST

Response Type

Tag

Parameters

  • Query: access_token String

    Samsara API access token.

  • Body: tagCreateParams TagCreate(TagCreate)

deleteTagById

Permanently deletes a tag.

URL

https://api.samsara.com/v1/tags/{tag_id}

HTTP Method

DELETE

Response Type

Parameters

  • Query: access_token String

    Samsara API access token.

  • Path: tag_id Long

    ID of the tag.

getAllTags

Fetch all of the tags for a group.

URL

https://api.samsara.com/v1/tags

HTTP Method

GET

Response Type

inline_response_200_9

Parameters

    Samsara API access token.

    Optional group ID if the organization has multiple groups (uncommon).

HTTP Method

Response Type

Parameters

    Samsara API access token.

  • Body: tagModifyParams TagModify(TagModify)

updateTagById

Update a tag with a new name and new members. This API call would replace all old members of a tag with new members specified in the request body. To modify only a few devices associated with a tag use the PATCH endpoint.

URL

https://api.samsara.com/v1/tags/{tag_id}

HTTP Method

PUT

Response Type

Tag

Parameters

  • Query: access_token String

    Samsara API access token.

  • Path: tag_id Long

    ID of the tag.

  • Body: updateTagParams TagUpdate(TagUpdate)
