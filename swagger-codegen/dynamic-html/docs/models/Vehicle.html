Vehicle

  • engineHours : Long
    Total engine hours for the vehicle.
  • fuelLevelPercent : Double
    The fuel level of the vehicle as a percentage. (0.0 to 1.0)
  • id : Long
    ID of the vehicle.
  • name : String
    Name of the vehicle.
  • note : String
  • odometerMeters : Long
    The number of meters reported by the odometer.
  • vin : String
    Vehicle Identification Number.
